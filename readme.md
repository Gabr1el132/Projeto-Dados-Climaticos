Esse programa foi desenvolvido como trabalho final para a matéria de Programação I, que aborda a linguagem C. Para usá-lo você deve inserir datas específicas sobre um estado específico e então adicionar as informações requeridas durante a execução do programa (temperatura, umidade, precipitação e velocidade do vento). Então a partir desses dados o programa exibe opções possíveis para realizar com estes dados, como por exemplo gerar relatórios mensais. Esses dados e relatórios são então salvos em arquivos de texto para possíveis análises ou mudanças futuras.
